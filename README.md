# anti-cheating

This web page demonstrates most of the actions that a web page can detect to tell if someone cheating.

## Supported Events

- Left/Returned to the Page
- Printed the Page
- Focus Switched
- Key Pressed
- Mouse Moved
- Window Size Changed
- Clicked
- Right Clicked
- Copied Data
- Cut Data
- Pasted Data

## Supported Media

- Camera
- Screen Sharing
